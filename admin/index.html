データ収集・分析パイプラインの実行状況と手動操作

パイプライン構成

毎日 04:00 JST に自動実行されるデータ収集・分析パイプライン
#ステップスクリプト説明頻度
1ニュース収集collect_news.py47+ RSSフィードからPESTLEニュース収集（各カテゴリ334件目標）毎日
2AI分析ai_analyze.py翻訳 + CLA分析 + 弱いシグナル検出毎日
3論文キュレーションcurate_daily_papers.pyOpenAlexから5分野の論文を選定・翻訳・新規性スコア毎日
4アラート検出detect_alerts.py異常・新出現トピック検出毎日
5日次レポートgenerate_daily_report.py日本版・グローバル版レポート生成（~5000字）毎日
6シナリオ生成generate_scenarios.py4シナリオ（不確実性-影響マトリクス）週次（月曜）
7歴史データ収集collect_historical_daily.py1990-2025の歴史的PESTLEデータ（1年/日）毎日
8履歴更新update_history.pypestle_history.json更新（50件/カテゴリ/四半期上限）毎日
9デプロイgit pushGitHub Pagesへデプロイ毎日

手動実行コマンド

ターミナルで実行するコマンド
全パイプライン実行
cd ~/projects/apps/future-insight-app && bash scripts/daily_update.sh
ニュース収集のみ
cd ~/projects/apps/future-insight-app && ~/.venv/bin/python scripts/collect_news.py
AI分析のみ
cd ~/projects/apps/future-insight-app && ~/.venv/bin/python scripts/ai_analyze.py
日次レポート生成のみ
cd ~/projects/apps/future-insight-app && ~/.venv/bin/python scripts/generate_daily_report.py
デプロイのみ（git push）
cd ~/projects/apps/future-insight-app && git add data/ && git commit -m "chore: manual data update $(date +%Y-%m-%d)" && git push
