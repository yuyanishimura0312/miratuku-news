パイプライン構成

#ステップスクリプト説明頻度
1ニュース収集collect_news.pyRSSフィード + GDELT APIからPESTLEニュース収集毎日
2AI分析ai_analyze.py翻訳 + CLA分析 + 弱いシグナル検出毎日
3論文キュレーションcurate_daily_papers.py5分野の論文を選定・翻訳・新規性スコア毎日

5日次レポートgenerate_daily_report.py日本版・グローバル版レポート生成毎日
6シナリオ生成generate_scenarios.py4シナリオ生成週次（月曜）

cd ~/projects/apps/future-insight-app && ~/.venv/bin/python scripts/collect_news.py
AI分析のみ
cd ~/projects/apps/future-insight-app && ~/.venv/bin/python scripts/ai_analyze.py
デプロイのみ
cd ~/projects/apps/future-insight-app && git add data/ && git commit -m "chore: manual update $(date +%Y-%m-%d)" && git push
